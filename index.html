Anaglyph method comparison. A new twist: anaglyphs on the fly

The purpose of this web page is to generate anaglyphs following the various methods exposed by Peter Wimmer on his article Anaglyph Methods Comparison.

Anaglyphs here are built on the fly using HTML5, CSS, JavaScript and some SVG filters that do the magic.

Type a link to a stereoscopic pair in the box below and click the button Anaglyph it!

The link by default is provided just in case you haven't got any on hand. It points to a stereoscopic pair by Pascal Martiné —I still have to ask him for his consent.

The code of this page is on repository https://github.com/raphikWasHere/anaglyph-method-comparison/
